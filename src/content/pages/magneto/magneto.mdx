## Magneto hard-fork

Ethereum Classic is undergoing a network upgrade code-named Magneto **around July 23rd 2021**.
Magneto is specified in [ECIP-1103](https://ecips.ethereumclassic.org/ECIPs/ecip-1103)
and brings the Ethereum Berlin protocol changes to ETC.
This is an important security upgrade and is not contentious.

**All node operators will need to update their clients before the activation block
to stay on the ETC mainnet chain.**



# Client releases with Magneto support

* [Core Geth v1.12.0](https://github.com/etclabscore/core-geth/releases/tag/v1.12.0) or later
   * As of 1st July, it appears that 0 of 387 Core Geth nodes have been updated (https://etcnodes.org) but that seems to be a bug in the crawler.  Both Antoine Toulme and ETC Core are working on new crawlers.
* [Hyperledger Besu v21.1.7](https://github.com/hyperledger/besu/releases/tag/21.1.7) or later
    * As of 1st July, it appears that 21 of 40 Besu nodes have been updated (https://etcnodes.org)
